软件特点

  • 注入: 打开, 等待, 注入!

  • 安全性: 注入后不会占用任何文件, 注入的数据大多是联网下载, 不留文件, 安全性较高!

  • 兼容性: 理论上能在所有正版Windows NT上运行, 不需要安装额外运行库!

使用方法

请勿在使用本软件时进入禁止使用修改模块的服务器 (?)

  1. 启动本软件并正确登录

  2. 待界面消失后启动游戏

  3. 注入完成

错误代码

  • Invalid password.

    密码无效, 请输入正确密码!

  • Outdated password.

    密码过期, 请输入有效密码!

  • Failed to get data from server.

    请确认网络是否正常!
    也有可能是DNS的问题!

  • Wrong server status.

    服务器问题, 请通知作者修复!
    也有可能是当前网络特别差!

  • Failed to inject.

    可能的原因有很多:
    被杀软拦截!
    目标进程被保护!
    目标进程在注入时退出(或崩溃)!
    罕见错误!

  • Failed to get disk SN.

    原因需要参考错误代码! (?)
    也许把软件放到另一个存储设备(本地磁盘, U盘, 移动硬盘等)能够解决!

  • Failed to create file.

    原因需要参考错误代码! (?)
    也许删除"password.cfg能解决

  • Failed to read file.

    原因需要参考错误代码! (?)
    也许重启电脑能解决

  • 校验错误.

    从服务器或者本地获取的数据未通过校验, 请确保网络连接稳定或重新下载离线数据包!

常见问题&解答
