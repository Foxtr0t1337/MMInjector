软件特点

  • 注入: 打开, 注入!

  • 安全性, 注入后不会占用任何文件, 注入的数据大多是联网下载, 不留文件, 安全性较高!

  • 兼容性, 理论上能在所有正版Windows NT上运行, 不需要安装额外运行库!

使用方法

请勿在使用本软件时进入禁止使用修改模块的服务器 (?)

  1. 启动本软件并正确登录

  2. 待界面消失后启动游戏

  3. 注入完成

FAQ

  • Invalid password.

    密码无效, 请输入正确密码!

  • Outdated password.

    密码过期, 请输入有效密码!

  • Failed to get data from server.

    请确认网络是否正常!
    也有可能是DNS的问题!

  • Wrong server status.

    服务器问题, 请通知作者修复!
    也有可能是当前网络特别差!

  • Failed to inject.

    可能的原因有很多:
    被杀软拦截!
    目标进程被保护!
    目标进程在注入时退出(或崩溃)!
    罕见错误!
